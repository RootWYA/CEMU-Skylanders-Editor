SkyEditor — Standalone Setup
Quick Start

    Run the App: Double-click SkyEditor.exe. You do not need to install Python or any other software to run it.

    Edit Your Save: Click "Open .sky file...", modify your Skylander's Gold, XP, or Nickname, and click Save.

    Automatic Backup: A .bak backup of your original file is created automatically the first time you save, ensuring your data stays safe.

What's Verified vs. Not
Verified (Tested & 100% Safe)

    Gold & Nickname modification.

    XP (Levels 1–10 tier).

    The AES-128/MD5 block encryption.

    Area-pair failover writing: The editor writes data exactly like an official portal does (by updating a backup slot first and then promoting it). If a save process is interrupted, your original file will not be corrupted.

Not Implemented (Not Supported Yet)

    Hats, Heroics, and Wow Pows.

    XP above Level 10: The offsets and checksum mechanics for higher levels are completely omitted to prevent file corruption.
